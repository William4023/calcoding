Practice Number Theory With C++

Tutor: William Shi
William is a fourth grader boy that like coding a lot. He like basketball, chess and math. He has two years of experience in programming and loves using using Studio, Blender, and Procreate to create projects.

What we’ll learn:
We'll learn how to use C++ to solve simple Number Theory problems like the number of primes less than N, the number of factors N has, and more. This serves as a small introduction to C++ for new people.

Target Student:
Students learning number theory

Text Books:
Introduction to Number Theory, AoPS
Guide to Competitive Programming, Antti Laaksonen

Syllabus:
Week 01: Input & Output 2.1.1
Week 02: Work with Numbers 2.1.2
Week 03: Vectors 5.1.1
Week 04: Common Time Complexities 3.1.2
Week 05: Backtracking 2.2.3
Week 06: Generating Permutations 2.2.2
Week 07: Primes & Factors 11.1.1
Week 08: Sieve of Eratosthenes 11.1.2
Week 09: Euclid’s Algorithm 11.1.3
Week 10: Modular Exponentiation 11.1.4
Week 11: Euler’s Theorem 11.1.5
Week 12: Solving Equations 11.1.6
